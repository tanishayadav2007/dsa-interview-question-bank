# DSA-Interview-Question-Bank
This repository contains my Data Structures and Algorithms (DSA) practice for interview preparation.

It includes a structured collection of problems based on real company questions such as TCS, Amazon, Microsoft, IBM, and more.

Topics Covered:
- Arrays
- Linked Lists
- Stack
- Queue

Each problem includes:
- Clean Java implementation
- Approach explanation
- Time and space complexity

This repository is part of my journey to strengthen problem-solving skills and prepare for technical interviews.
